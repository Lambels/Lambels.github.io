Porting panama to C++: concurrency patterns from a channel

I found the crust of rust series from Jon Gjengset to be a priceless tool for learning low level programming paired with concurrency primitives. Through my study of the guide I wanted to port a lot of the things written there to C++. This was in an effort to better understand C++ but also to better appreciate the footguns that Rust natively avoids using the borrow checker.
