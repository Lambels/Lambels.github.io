üèó Projects: #

These projects are the result of 3 years of working with golang. The order represents the quality of the projects (decreasing order) as I see them.

Sinoname: #

Sinoname is a highly customizable, flexible, extensible, out of the box ready, pipeline for generating similarly looking strings. It is very good in generating similar looking usernames when the initial username is not available.

Cronjob: #

Cronjob is a cron like scheduler which uses golang time specification for scheduling jobs. Cronjob has a very clean and intuitive api which provides great user experience.

Relationer: #

Relationer provides a real-time interface to view simple relationships (graph relationships) between people. The relationer server produces messages which are sent to a centralized message broker to be consumed by you on either: the relationer cli receiving point, provided frontend, or your own customizable app.

Autho: #

Autho is a customisable and clean implementation of all OAuth and OAuth2.0 web providers with full implementation for popular providers such as twitter, facebook, github and google. Autho helps you focus on the logic side of your application whilst providing an idiomatic api to handle the boilerplate.
